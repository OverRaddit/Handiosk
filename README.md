# Handiosk

- 실행법

```shell
# 리액트 앱 패키지 다운
npm i

# 파이썬 스크립트 실행(가상환경 셋팅필요.)
cd PythonScript
python mydual.py
cd ..

# 리액트 앱 실행
npm start
```
